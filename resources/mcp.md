# MCP Servers

**Model Context Protocol Servers** are the lifeblood to real action and information in large langauge models.

There are a lot out there, but where do you find them? The answer is that this standard is decentralized and anyone is free to distribute it however they like. It can range from being locally executed to requesting from a remote service.

Below are some resources to help find them, and also some personal favorite MCP servers.

# The main website for MCP

https://modelcontextprotocol.io/

# Find MCP Servers

- [Official MCP Registry](https://registry.modelcontextprotocol.io/)
- [The modelcontextprotocol GitHub](https://github.com/modelcontextprotocol/servers)
- [Awesome MCP Servers](https://mcpservers.org/)
- [Github's repository](https://github.com/mcp)
